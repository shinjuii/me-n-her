# 📸 Folder Foto Cleo & Luna

Taruh foto-foto kalian di sini!

## Cara pakai:
1. Taruh foto di folder ini
2. Di `index.html`, ganti bagian `<div class="polaroid-img placeholder-img">` dengan:

```html
<div class="polaroid-img">
  <img src="images/nama-foto.jpg" alt="deskripsi foto" />
</div>
```

## Struktur yang disarankan:
- `gaming/` — foto-foto saat main game
- `random/` — momen random sehari-hari  
- `art/` — karya seni kalian

Enjoy! 💕
